# Project Evidence

## AI systems for UK planning (Xylo)
Sam is building AI systems for planning permission workflows—a domain combining real estate/property intelligence, document processing, and regulatory complexity.

The work spans full-stack product delivery: PDF extraction with AI-generated markdown, multi-turn chat systems with tool-based response grounding, ML evaluation frameworks (Braintrust) for response quality, and background job processing for async workflows. The architecture uses a pnpm monorepo (30+ packages) with Turbo for build orchestration, AWS Lambda for serverless compute, Supabase for data, and Inngest for workflow automation.

Key technical contributions include: document extraction pipelines, policy analysis via LLMs, multi-turn conversation evals with Braintrust, relationship/reference extraction between documents, and 2FA authentication flows. The work prioritizes reliable AI outputs, proper ML evals to catch regressions, and incremental feature delivery in a complex regulatory domain.

## Carbon accounting platform
Sam built product and UX capabilities for carbon accounting workflows used by Private Equity firms. The system focused on portfolio-level emissions visibility and reliable reporting workflows.

Technical emphasis included event-driven synchronization, robust backend services, and frontend experiences designed for decision support rather than generic dashboards.

## Enterprise payments modernization

The architecture prioritized operational reliability, clear service boundaries, and production-safe rollout strategies.
